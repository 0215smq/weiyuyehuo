# 韩松传

### 关于《韩松传》

女主穿越到陌生的时代，发现天下大乱，烽烟四起。

而自己文不通军事历史，理不懂金融科技，遇事想要跑路，还分不清南北。

好在身边土著都是大佬，可以学习一下。
